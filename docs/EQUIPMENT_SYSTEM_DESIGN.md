## 🎯 装备获取途径

### 1. 商店购买
- **凡品全部装备**（15件）
- **仙品部分装备**（前20件，8-11级）

### 2. 副本掉落
- 低级副本：凡品装备
- 中级副本：仙品装备（8-12级）
- 高级副本：仙品稀有装备（12-14级）

### 3. 任务奖励
- 主线任务：青竹蜂云剑、凤凰羽等稀有仙品
- 每日任务：装备强化材料
- 成就任务：特殊饰品

### 4. 积分商城兑换（神品套装）
- **四象套装**：通过积分商城购买
  - 每件装备需要14,000-24,000积分
  - 每人每件限购1次
  - 需要15级以上才能购买
  - 完整套装收集可激活2/4/6件套装效果
