# 积分商城系统说明

## 概述

积分商城是一个基于积分兑换的特殊商城系统，玩家可以通过游戏内的各种活动获得积分，然后使用积分兑换珍稀物品。

## 核心特性

### 1. 独立货币系统
- **积分（Credits）**: 独立于灵石的特殊货币
- 只能通过特定活动获得，不可交易
- 专用于积分商城兑换

### 2. 多样化商品分类
- 📖 **功法**: 极品功法，大幅提升修炼速度
- ⚔️ **法宝**: 强力法宝，包括《凡人修仙传》经典物品
- 💊 **丹药**: 突破丹药，提升成功率
- ✨ **特殊物品**: 稀有道具，独特效果

### 3. 完善的限购机制
- **总限购**: 每个商品设置终身购买次数上限
- **每日限购**: 某些商品每日限购次数
- **库存控制**: 限量商品，售完即止
- **境界要求**: 高级商品需达到指定境界

## 积分获取途径

### 每日活动
- 📅 **签到**: 10积分基础 + 连续签到奖励（最多+35/天）
- 🎯 **每日任务**: 5-50积分（根据难度）

### 战斗系统
- ⚔️ **竞技场胜利**: 5积分/场 + 连胜奖励
- 🐲 **世界BOSS**: 10-50积分（根据排名）

### 成长奖励
- 🌟 **境界突破**: 5-200积分（根据境界）
  - 炼气期: 5积分
  - 筑基期: 20积分
  - 结丹期: 50积分
  - 元婴期: 100积分
  - 化神期: 200积分

### 成就系统
- 🏆 **成就完成**: 10-100积分（根据等级）
  - 铜牌: 10积分
  - 银牌: 25积分
  - 金牌: 50积分
  - 铂金: 100积分

### 其他途径
- 🏛️ **宗门贡献**: 参与宗门活动获得
- 🎁 **活动奖励**: 特殊活动期间
- 👑 **管理员赠送**: 特殊情况下的补偿

## 使用方法

### 浏览商城
```
/积分商城 或 /商城
```
显示商城主菜单，包含所有商品分类。

### 查看积分
```
/我的积分
```
显示当前积分余额、获得/消耗统计和最近交易记录。

### 兑换商品
1. 使用 `/积分商城` 打开商城
2. 选择商品分类（功法/法宝/丹药/特殊）
3. 点击商品查看详情
4. 点击"立即兑换"按钮
5. 系统自动扣除积分并发放商品

## 交易记录

系统会自动记录所有积分变动：
- ✅ 记录时间
- ✅ 变动数量（+ 获得 / - 消耗）
- ✅ 变动原因
- ✅ 变动前后余额

## 购买限制说明

### 库存限制
- **限量商品**: 全服共享库存，售完即止
- **无限商品**: 标记为"无限"，随时可购买

### 购买次数限制
- **终身限购**: 每个玩家一生只能购买指定次数
- **每日限购**: 每天可购买的次数，每日0点重置

### 条件限制
- **境界要求**: 必须达到指定境界才能购买
- **等级要求**: 境界内的小等级要求（如筑基期5层）
- **VIP要求**: 部分商品需要VIP等级（待实现）

## 常见问题

### Q: 积分可以交易吗？
A: 不可以。积分是账号绑定的，不可转让、不可交易。

### Q: 限购次数会重置吗？
A: 终身限购不会重置，每日限购每天0点重置。

### Q: 买错了可以退货吗？
A: 目前不支持退货。请仔细确认后再购买。

### Q: 积分会过期吗？
A: 积分永久有效，不会过期。

### Q: 如何快速获得大量积分？
A:
1. 坚持每日签到
2. 完成每日任务
3. 参与竞技场
4. 击杀世界BOSS
5. 完成高难度成就

### Q: 小绿瓶值得买吗？
A: 绝对值得！这是游戏中最强力的法宝之一，但价格昂贵（15,000积分）且限量3份，建议尽早购买。
