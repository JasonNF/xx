## 🎨 HBO Max 设计风格特点

### 视觉特征
- **深色主题**：黑色/深紫色背景
- **渐变色卡片**：紫色到蓝色的渐变
- **大图标**：突出视觉元素
- **流畅动画**：页面切换和元素加载动画
- **极简布局**：减少视觉干扰
- **沉浸式体验**：全屏内容展示

### 导航特点
- **侧边栏导航**（桌面端）
- **底部标签栏**（移动端）
- **面包屑导航**
- **快速搜索**
- **用户头像下拉菜单**

## 🔄 页面切换动画

### 路由转场
```css
/* HBO 风格页面切换 */
.page-enter-active,
.page-leave-active {
  transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
}

.page-enter-from {
  opacity: 0;
  transform: translateX(30px);
}

.page-leave-to {
  opacity: 0;
  transform: translateX(-30px);
}
```

### 元素加载
```css
/* 卡片依次加载 */
.card-stagger-enter-active {
  transition: all 0.5s ease;
  transition-delay: calc(var(--index) * 0.1s);
}

.card-stagger-enter-from {
  opacity: 0;
  transform: translateY(20px);
}
```
