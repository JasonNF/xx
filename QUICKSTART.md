# 快速开始指南

## 🚀 5分钟快速部署

### 步骤1：获取Bot Token

1. 在Telegram搜索 `@BotFather`
2. 发送 `/newbot` 创建机器人
3. 按提示设置名称，例如：
   - Bot name: `修仙世界`
   - Bot username: `xiuxian_world_bot`（必须以_bot结尾）
4. 复制获得的Token（格式类似：`123456789:ABCdefGHIjklMNOpqrsTUVwxyz`）

### 步骤2：配置环境

```bash
cd xiuxian-game
cp .env.example .env
```

编辑 `.env` 文件，粘贴您的Token：

```env
BOT_TOKEN=你的Bot_Token
BOT_USERNAME=xiuxian_world_bot
```

### 步骤3：安装依赖

**方式A：使用pip**
```bash
pip install python-telegram-bot sqlalchemy alembic aiosqlite pydantic pydantic-settings python-dotenv apscheduler redis
```

**方式B：使用项目配置**
```bash
pip install -e .
```

### 步骤4：初始化数据

```bash
python scripts/init_data.py
```

这会创建：
- 怪物数据（野狼、狐妖、Boss等）
- 物品数据（武器、护甲、丹药等）
- 功法数据（五行诀、紫霄神功等）

### 步骤5：启动游戏

```bash
./run.sh
```

或者：

```bash
cd src
python -m bot.main
```

### 步骤6：开始游戏

1. 在Telegram搜索您的Bot用户名（例如：`@xiuxian_world_bot`）
2. 点击Start或发送 `/start`
3. 开始您的修仙之旅！

## 📋 基础游戏流程

### 新手引导

1. **创建角色** - `/start` 自动创建
2. **查看状态** - `/status` 查看角色信息

## 🔧 常见问题

### Q: Bot没有响应？
A: 检查以下几点：
1. Bot Token是否正确
2. 网络连接是否正常
3. 查看控制台是否有错误信息

### Q: 数据库文件在哪里？
A: 默认在 `data/xiuxian.db`

### Q: 如何重置游戏数据？
A: 删除 `data/xiuxian.db` 文件，重新运行 `init_data.py`

### Q: 修炼时间到了怎么没通知？
A: 需要手动执行 `/finish` 命令完成修炼

### Q: 如何修改游戏参数？
A: 编辑 `.env` 文件中的配置项
